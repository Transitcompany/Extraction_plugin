### Common Issues & Solutions:

**If commands don't work:**
1. Restart the server with the new jar
2. Check console for any errors
3. Ensure permissions are set correctly

**If GUI doesn't open:**
1. Check player has extraction permission
2. Verify server isn't in offline mode
3. Check if player inventory is full

**If stats don't display:**
1. Player data initializes on first use
2. Check playerdata.yml file exists
3. Verify plugin has write permissions
